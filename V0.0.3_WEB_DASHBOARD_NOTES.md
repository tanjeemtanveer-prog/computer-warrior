# Computer Warrior v0.0.3 — Web Dashboard

## Purpose

v0.0.3 adds a browser UI without changing what Computer Warrior collects. The
Python process remains the sole input tracker. The web page receives only its
already-aggregated XP snapshot.

## Local API boundary

- Host: `127.0.0.1` only.
- `GET /api/stats`: aggregate Session, Today, Lifetime, level, and next-reward progress.
- `GET /api/health`: simple local availability check.
- `POST /api/control/pause`, `/resume`, `/exit`: local controls for the existing tracker.
- No CORS headers are sent and the page is served by that same local process.

The API has no typed keys/text, key identities, mouse coordinates, click
locations, app names, window titles, URLs, clipboard values, screenshots, or
event timestamps.

## Manual QA

1. Run `run_self_tests.bat` and confirm every test passes.
2. Run `launch_web_dashboard.bat`; the browser should open at `127.0.0.1:8765`.
3. Press regular keys, click, move, and scroll; the page changes within about half a second.
4. Open `Details`; verify each Today and Session category matches the CLI dashboard.
5. Press Pause in the web page, generate input, and verify XP does not change.
6. Press Resume, generate input, and verify XP resumes.
7. Press F9 and verify the browser changes between TRACKING ACTIVE and TRACKING PAUSED.
8. Press F10 or Exit and confirm the CLI reports that XP was saved successfully.
