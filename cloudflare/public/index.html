THE PRIVACY PROMISE

Progress tracking without activity surveillance.

Computer Warrior is designed around the smallest useful data boundary. The Windows tracker calculates totals locally. Account sync, when enabled, sends only four aggregate XP numbers in ordered batches.

HOW IT WORKS

Three steps. One clear boundary.

The tracker remains useful without an internet connection. Online features are an optional layer, not a requirement.

  1. 1

    Run the Windows tracker

    Computer Warrior listens only for physical input signals and ignores injected or synthetic events.

  2. 2

    Build progress locally

    Your dashboard, daily mission, focus quests, and lifetime XP continue working offline on your PC.

  3. 3

    Choose whether to sync

    Sign in only if you want multi-device totals or the opt-in leaderboard. The browser never receives the desktop session token.
