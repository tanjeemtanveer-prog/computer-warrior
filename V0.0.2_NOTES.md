# Computer Warrior v0.0.2

## Purpose

Make the already-tested XP categories visible before beginning GUI or background-startup work.

## Added

- Live table for Keyboard, Mouse clicks, Cursor movement and Scrolling XP.
- Session, Today and Lifetime columns for every category.
- Total XP row below the activity rows.
- Cursor-pixel and scroll-step progress toward the next XP.
- Visible RUNNING/PAUSED state.
- In-place Windows console refresh with a safe non-interactive fallback.
- Dashboard-formatting and redraw regression tests.
- Explicit v0.0.1-hotfix.1 stats compatibility test.

## Preserved

- Stats schema version 1.
- `%LOCALAPPDATA%\ComputerWarrior\stats.json` and its backup.
- All XP rules and remainders.
- pynput 1.8.2 compatibility and injected-event filtering.
- F9 pause/resume and F10 save/exit.
- Atomic save, backup recovery, listener-health monitoring and single-instance protection.
- Privacy boundary: aggregate XP only; no input content or activity history.
