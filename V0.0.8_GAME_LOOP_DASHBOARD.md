# Computer Warrior v0.0.8 — Game Loop Dashboard

## Purpose

Make the existing local dashboard feel more like a game without widening the
privacy boundary or changing XP rules.

## Changes

- Derives a visible rank title from the already-visible XP level.
- Shows a live session-pulse badge next to level progress.
- Adds a percentage Activity Mix inside the existing collapsible activity
  drawer. It is calculated only from today's keyboard, click, cursor and
  scroll XP totals.
- Improves the online leaderboard's visual hierarchy.

## Privacy

No new metrics, event histories, key identities, typed text, cursor positions,
window names, websites, clipboard data, or screenshots are collected or sent.

## Deployment

This is a local dashboard release. Run local QA, commit and push the source.
Do not run a D1 migration or redeploy the Worker.

## Manual QA

1. Launch the local dashboard and confirm the rank chip and session badge
   change as XP changes.
2. Open **Details** and verify the four-part Activity Mix matches the four
   displayed daily XP values.
3. Open **Online** and verify the leaderboard still shows the account total.
4. Pause and resume to confirm live tracking controls still work.
