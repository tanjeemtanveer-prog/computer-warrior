# Computer Warrior v0.0.9.1 — Daily Momentum chart fix

## Change

The seven-day signal now distinguishes a date with no saved local aggregate
history from a date with activity. Missing dates render as an empty dash with
the accessible label `No data`; they no longer appear as full or minimum-height
activity bars.

## Privacy and compatibility

- The added `has_data` field is a local dashboard presentation flag.
- It does not contain input content, identities, positions, or event history.
- Existing schema-version-1 stats files load unchanged; no XP is changed.
- No Cloudflare Worker deployment, D1 migration, or secret change is needed.

## Verification

Run `verify_project.bat`. The Python tests verify the local payload marks
missing history dates, while the dashboard source test verifies the empty-state
rendering path.
