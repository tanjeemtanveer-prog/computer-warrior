# Computer Warrior v0.0.9 — Daily Momentum

## Purpose

Turn existing aggregate XP into a clearer daily game loop without inferring
what the person is doing on their computer.

## Changes

- A configurable daily goal from 50 to 50,000 XP, stored locally.
- A goal progress card, completion message and streak counter.
- A rolling seven-day visual history of local date and aggregate XP total.
- Derived badges: First Spark, Goal Cleared, Momentum and Five Figures.

## Privacy

The new data is limited to the daily target, ISO calendar date and one total XP
number per day. It does not store key identities, typed text, cursor
coordinates, websites, windows, clipboard data, screenshots, or input-event
history. It is not included in Worker sync requests.

## Compatibility

Existing schema-version-1 stats files load safely with a default 500-XP goal
and empty history. The first day rollover begins the local history.

## Deployment

Run `verify_project.bat`, commit and push. Do not migrate D1, rotate secrets,
or redeploy the Worker.
